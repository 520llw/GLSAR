## 🎯 项目简介

GLSAR专注于**SAR（合成孔径雷达）图像的大规模识别和分类**，主要特点：

- 📊 大规模图像处理
- 🎯 高精度目标识别
- 🔄 支持水平SAR图像
- 🚀 快速训练和推理

---

## ✨ 功能特点

- ✅ 大规模SAR图像数据集处理
- ✅ 深度学习目标检测
- ✅ 数据增强和预处理
- ✅ 可视化训练过程
- ✅ 模型评估和测试

---

## 🚀 快速开始

```bash
git clone https://github.com/520llw/GLSAR.git
cd GLSAR
pip install -r requirements.txt
python train.py
```
